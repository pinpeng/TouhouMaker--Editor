# TouhouMaker--Editor

东方弹幕编辑器（车万没课儿）的编辑器模块

## 编译环境

Qt Creator 5.0.2 (Community) 

MinGW 7.3.0 32-bit

## 使用方法

目前使用方法是，在编译好的二进制文件目录下新建一个debug文件夹，之后将[TouhouMaker-Interpreter](https://github.com/pinpeng/TouhouMaker-Interpreter)产生的编译文件放入该目录中，后续将考虑优化方案。
